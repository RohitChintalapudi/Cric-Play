# Cric Play - Mini Cricket Game in React

Cric Play is a fun and interactive mini cricket game built using React.js. Each player gets to bat, and every player faces 6 balls. The game keeps track of scores, determines the winner, and allows users to restart the match with a single click.

---

## Features

- Turn-based gameplay with two players
- Fully mobile responsive design with premium aesthetics
- Each player gets 6 balls to bat
- Live Target tracking for the second player
- Score tracking for each player and team
- Hit History log to track recent performance
- Winner is declared automatically based on total scores
- "Play Again" button to reset and restart the match
- Built with React functional components, context API, and hooks

---

## Tech Stack

- React.js - Frontend framework
- JavaScript (ES6+) - Game logic and interactions
- CSS3 - Structure, styling, and glassmorphism
- Vite - Build tool

---

## Installation and Setup

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm run dev` to start the development server
4. Run `npm run build` to create a production-ready build
