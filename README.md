# Cric Play — Interactive Mini Cricket Game in React

Cric Play is a modern and engaging mini cricket game built using React.js that delivers a smooth turn-based cricket experience directly in the browser. Designed with a premium UI and responsive gameplay mechanics, the game allows two players to compete in a fast-paced 6-ball cricket challenge.

Players can score runs, chase targets, track match history, and instantly replay matches with an immersive and mobile-friendly experience.

# How the Game Works

## Match Flow

1. Player 1 starts batting
2. Player 1 faces 6 balls
3. Total score is recorded
4. Player 2 begins chase
5. Live target is displayed
6. Player 2 faces 6 balls
7. Final scores are compared
8. Winner is automatically declared

---

## Scoring System

Runs are randomly generated on every shot:

| Outcome | Runs |
|--------|------|
| Dot Ball | 0 |
| Single | 1 |
| Double | 2 |
| Triple | 3 |
| Boundary | 4 |
| Six | 6 |
